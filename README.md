# BataoGPT – AI-Powered Conversational Chat Application

BataoGPT is a full-stack AI chatbot application inspired by ChatGPT, designed to provide a seamless and interactive conversational experience. The platform enables users to engage in real-time AI-powered conversations while maintaining persistent chat history for future reference. Built with a modern web development stack, the application focuses on performance, scalability, and an intuitive user experience.

The frontend is developed using React.js and Vite, providing a fast and responsive interface with reusable, component-based architecture. Users can interact with the AI through a clean chat interface that supports Markdown-formatted responses, allowing rich text rendering, lists, tables, and other structured content. To enhance readability for technical discussions, the application also supports syntax-highlighted code blocks, making programming-related responses easier to understand.

On the backend, Node.js and Express.js handle API requests, business logic, and communication with AI models through OpenRouter APIs. The backend is structured to ensure maintainability and scalability while efficiently managing conversation workflows. MongoDB is used for data persistence, enabling users to store and retrieve previous conversations, creating a continuous chat experience across sessions.

To improve user engagement, the application incorporates typing animation effects that simulate real-time AI response generation, creating a more natural conversational flow. The project emphasizes clean code practices, reusable components, and modular architecture, making future feature additions and maintenance straightforward.

---

### Key Features

* Real-time AI-powered conversational interface
* Integration with OpenRouter APIs for AI model access
* Persistent chat history using MongoDB
* Markdown-based response rendering
* Syntax-highlighted code blocks for technical content
* Typing animation effects for enhanced user experience
* Reusable React components and modular architecture
* Responsive design for desktop and mobile devices
* Structured backend workflows using Express.js and RESTful APIs

---

### Tech Stack

**Frontend:** React.js, Vite, JavaScript, CSS

**Backend:** Node.js, Express.js

**Database:** MongoDB

**AI Integration:** OpenRouter API
